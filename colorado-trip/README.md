# 🏔️ Colorado 2026 – Trip Assistant

A bilingual (Hebrew/English) AI-powered trip assistant for our Colorado family road trip, June 30 – July 8, 2026.

## Features

- 💬 Chat with Claude AI about the trip plan
- 🗓️ Select specific days (multi-select)
- 🎒 Quick queries: gear, weather, route, drive times, restaurants, grocery lists
- 🗺️ Google Maps links in every answer
- 📂 Upload updated trip plan file
- 🇮🇱 / 🇺🇸 Hebrew / English toggle
- 📱 Mobile-friendly PWA-ready design
